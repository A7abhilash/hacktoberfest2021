## 🛠️ Installation Steps

1. Clone the repository

```bash
git clone https://github.com/javascript-spec/hacktoberfest-2020.git
```

2. Change the working directory

```bash
cd hacktoberfest-2020
```

3. Install dependencies

```bash
npm install
```

4. Run the app

```bash
npm start
```
